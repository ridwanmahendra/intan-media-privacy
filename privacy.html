1. Informasi yang Kami Kumpulkan

Aplikasi ini adalah alat otomatisasi pribadi yang hanya digunakan oleh pemilik akun Intan Media sendiri. Kami tidak mengumpulkan atau menyimpan data pribadi pengguna lain. Satu-satunya data yang diproses adalah:

  • Token Akses Pinterest: Disimpan secara lokal di server pribadi untuk keperluan otentikasi API.
  • Data Produk Afiliasi: Informasi produk (nama, deskripsi, gambar) yang diambil dari file Excel pribadi untuk diposting ke Pinterest.

2. Penggunaan Data

Data yang diproses hanya digunakan untuk:

Token akses dan konfigurasi aplikasi disimpan secara lokal di lingkungan server pribadi yang aman. Aplikasi ini tidak membagikan data dengan pihak ketiga mana pun.
